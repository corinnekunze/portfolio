Planet Argon 2013 Year in Review

I took on the Year in Review project for 2013 for Planet Argon. With some help from the team, I lead development, design, and production for the site. Notable technologies: CSS3 Animations, Javascript and Jquery, Sass, Responsive Development.

Planet Argon Timeline

A very cool project to tell the whole story of Planet Argon as a company. My role was everything from collecting the information, to design, to all the development. It was a great project to learn from. I was manipulating a large Javascript plugin (timeline.js) to behave like I wanted it to. Notable technologes: JSON, Javascript, Git submodules, Sass.

24 Hour Time Converter

Javascript is awesome, it's not every day that I get to write a javascript program from scratch. That's exactly what this project was. I wanted to use native javascript without a library to see what I could do. I built a converter that converts time both ways, using REGEX validation and user errors. Also used bootstrap to quickly make add some visual appeal.
